계란후라이🍳

레시피

  1. 프라이펜을 예열시켜요
  2. 식용유를 둘러요
  3. 계란을 까요
  4. 잘 구워봐요

Tip

  • 약불에 구워요(센 불에 하면 눌러붙거나 탑니다)
  • 운 좋으면 쌍란이 두개지요🥚🥚

김치볶음밥🍲

레시피

  1. 재료를 준비해요 (저는 귀찮아서 큼직하게 썰었어요)
  2. 식용유를 둘러요
  3. 김치를 먼저 볶아줘요
  4. 나머지 재료를 넣어요
  5. 밥을 넣고 잘 볶아봐요

Tip

  • 유튜브 보세요 (저도 잘 몰라요😅)

블루베리스무디🫐

레시피

  1. 냉동블루베리를 사요
  2. 바나나를 까서 냉동실에 얼려요 (바나ㅏ나ㅏㅏㅏ🍌)
  3. 얼린 블루베리와 바나나를 믹서기에 넣어요
  4. 우유를 넣고(200ml정도?) 요구르트도 반병 넣어요
  5. 믹서기로 잘 갈아봐요

Tip

  • 냉동실에서 꺼내서 살짝 녹이고 갈아야 잘 갈려요
  • 바나나보다 블루베리가 훨씬 많아야 해요
  • 너거들 블루베↑리↓스↓무↑디↓무봤나?

밀크티🧋

레시피

  1. 태국에서 티백을 사와요 (차트라뮤 존맛)
  2. 끓는 물을 사아아알짝 넣어서 티백을 우려요
  3. 꿀을 넣고 저어줘요
  4. 우유를 기호에 맞게 넣어요
  5. 얼음까지 띄워주면 완성!!

Tip

  • 꿀을 생각보다 많이 넣어야 맛있어요
  • 태국 또 가고싶다...🇹🇭🇹🇭🇹🇭
